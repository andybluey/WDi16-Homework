What is Perl?



Perl is a family of high-level, general purpose, interpreted, dynamic programming languages, including Perl 5 and Perl 6.


Although Perl is not officially an acronym, one of the most well known "backcronyms" defines it as Practical Extraction and Reporting Language.


It was originally designed for text manipulation and is now widely used for system administration, web development, network programming and GUI development.
